导出引擎（engine/export）

分页提取 → 分块写入临时文件 → 合并为最终文件

构造函数与配置

type ExportEngine struct {
    src        phys.DataSource
    outputDir  string        // 块文件和最终文件的输出目录
    pageSize   int           // 每页行数
    chunkPages int           // 每个块包含的页数
}

func New(src phys.DataSource, outputDir string, opts ...ExportOption) *ExportEngine

type ExportOption func(*ExportEngine)
func WithPageSize(n int) ExportOption       // 默认 1000
func WithChunkPages(n int) ExportOption     // 默认 10 (每块 10000 行)

Compensate 补偿逻辑

当前 Phase 补偿动作
running 截断当前块文件（chunk_<idx>.tmp）到 CurrentChunkSize（= LSN 在当前块内的分量）
merging 截断最终文件 output.dat 到 FinalFileSize（= CheckpointLSN）

Progress 映射

阶段 计算方式
running min(90, CurrentChunkIdx * 100 / TotalChunks) — 提取占 0-90%
merging 90 + MergedChunkIdx * 10 / TotalChunks — 合并占 90-100%

CheckpointLSN 语义

导出场景中 CheckpointLSN = 已安全写入磁盘的总字节数。

running 阶段：LSN 是所有已完成块的总字节数 + 当前块的安全字节数。
merging 阶段：LSN 是最终文件的当前安全字节大小。

恢复时 Compensate 通过截断文件到这个字节边界来消除脏数据。
