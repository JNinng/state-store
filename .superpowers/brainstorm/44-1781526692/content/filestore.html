Save 原子写入流程

1. 写入 <taskID>.state.tmp（完整序列化后的字节流）
2. 调用 tmpFile.Sync()，确保数据落入物理磁盘
3. 调用 os.Rename(tmpPath, finalPath) 将 .tmp 重命名为 .state
4. （可选）对父目录调用 dir.Sync()，保证目录元数据持久化
原子性保证：POSIX 同文件系统下的 rename(2) 是原子操作。 中途崩溃的场景：要么 .tmp 存在（未完成写入，重启后清理），要么 .state 存在（完整数据）。 绝不出现 .state 包含半写数据。

Load / Delete

func (r *FileRepository) Load(ctx context.Context, taskID string) ([]byte, error) {
    data, err := os.ReadFile(r.path(taskID))
    if os.IsNotExist(err) {
        return nil, nil  // Spec 要求：不存在 = nil, nil
    }
    return data, err
}

func (r *FileRepository) Delete(ctx context.Context, taskID string) error {
    err := os.Remove(r.path(taskID))
    if os.IsNotExist(err) {
        return nil  // 幂等：已删除的不报错
    }
    return err
}
