导出引擎（engine/export）— 修订版

分页提取 → 分块写入临时文件 → 合并为最终文件

构造函数与配置

type ExportEngine struct {
    src        phys.DataSource
    outputDir  string        // 输出目录
    outputFile string        // 最终文件名，如 "result.dat"
    pageSize   int           // 每页行数
    chunkPages int           // 每个块包含的页数
}

func New(src phys.DataSource, outputDir, outputFile string, opts ...ExportOption) *ExportEngine

文件命名规则

分块文件 = 最终文件名 + 块编号后缀，确保同一目录下多个导出任务互不干扰。 
outputDir/
├── result.dat.chunk_0.tmp    ← 第 0 块
├── result.dat.chunk_1.tmp    ← 第 1 块
├── result.dat.chunk_2.tmp    ← 第 2 块
├── result.dat                ← 合并后的最终文件
└── other-export.dat.chunk_0.tmp  ← 另一个导出任务，不冲突
命名模式：<outputFile>.chunk_<N>.tmp
